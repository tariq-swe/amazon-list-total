# Amazon List Total
Chrome extension that displays the total price of items in your Amazon wishlists and registries.

## Installation
1. Download or clone this repository
2. Open Chrome and navigate to chrome://extensions/
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the extension directory

## Usage
Navigate to any Amazon wishlist or registry page. The total will appear automatically in the bottom-left corner showing the sum of all item prices and item count.

### Supported URLs
- Wishlists: amazon.com/hz/wishlist/*
- Registries: amazon.com/gp/registry/*
- Lists: amazon.com/*/list/*
